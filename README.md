# homebrew-tap

Provides the following formulae:

- `libhydrogen`

## How do I install these formulae?

`brew install 0f-0b/tap/<formula>`

Or `brew tap 0f-0b/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "0f-0b/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check
